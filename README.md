# Project Iris — Iris Biometric Authentication System

> **FYP2:** Improving Image Detection for Biometric (Iris) Authentication Systems

A deep learning-based iris recognition system using a CNN (MiniIrisXception / IrisNet) trained with ArcFace loss, benchmarked against a classical Gabor filter baseline.

---

## Step 1: Download the Datasets

This project uses four subsets of the **CASIA-IrisV4** dataset. You must download them manually from Kaggle and place them in the correct directories.

> You will need a free [Kaggle account](https://www.kaggle.com) to download these datasets.

### Dataset Links

| Subset | Kaggle URL |
|---|---|
| CASIA-Iris-Interval | https://www.kaggle.com/datasets/swoyam2609/casia-iris-interval |
| CASIA-Iris-Lamp | https://www.kaggle.com/datasets/swoyam2609/casia-iris-lamp |
| CASIA-Iris-Thousand | https://www.kaggle.com/datasets/sondosaabed/casia-iris-thousand |
| CASIA-Iris-Syn *(replaces Twins)* | https://www.kaggle.com/datasets/monareyhanii/casia-iris-syn |

### Extraction Instructions

After downloading each dataset as a `.zip` archive, extract its contents into the corresponding folder:

```
data/raw/CASIA-Iris-Interval/   ← contents of the Interval download
data/raw/CASIA-Iris-Lamp/       ← contents of the Lamp download
data/raw/CASIA-Iris-Thousand/   ← contents of the Thousand download
data/raw/CASIA-Iris-Syn/        ← contents of the Syn download
```

The `data/raw/` directory is excluded from version control via `.gitignore` — do not commit image data.

## Step 2: Environment Setup

This project uses [uv](https://github.com/astral-sh/uv) for fast, reproducible Python environment management.

### Install uv (if not already installed)

```bash
pip install uv
# or on Windows via PowerShell:
# irm https://astral.sh/uv/install.ps1 | iex
```

### Install Python 3.9

```bash
uv python install 3.9
```

### Create the Virtual Environment

```bash
uv venv --python 3.9
```

### Activate the Environment

```bash
# Windows (PowerShell)
.venv\Scripts\Activate.ps1

# Windows (CMD)
.venv\Scripts\activate.bat

# Linux / macOS
source .venv/bin/activate
```

### Install Dependencies

```bash
uv pip install -r requirements.txt
```

> **Note:** This installs `tensorflow-cpu==2.10` plus `tensorflow-directml-plugin` for AMD GPU acceleration via DirectML on Windows. Do **not** install `tensorflow-gpu` — it is incompatible with AMD hardware on this stack.

### Verify Installation

```bash
python -c "import tensorflow as tf; print(tf.__version__)"
# Expected: 2.10.x
```

## Evaluation Metrics

| Metric | Description |
|---|---|
| FAR | False Acceptance Rate |
| FRR | False Rejection Rate |
| TAR | True Acceptance Rate (= 1 - FRR) |
| EER | Equal Error Rate |
| Accuracy | Overall classification accuracy |
